Frequently Asked Questions

Q: Is the app secure?

A: Yes, we use industry-standard encryption algorithms, and all data is stored locally without uploading to any server.

Q: Which two-factor authentication services are supported?

A: Supports all TOTP-based two-factor authentication services, including Google, Microsoft, Steam, Battle.net, and more.

Q: How do I backup my accounts?

A: The app supports exporting backup files, which you can save to a secure location for recovery when needed.

Q: Does the app require an internet connection?

A: The app requires an internet connection to synchronize time and ensure verification code accuracy. However, verification code generation itself does not require internet.
